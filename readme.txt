== Description ==
 
This plugin will create a new shortcode: [booking_activities_sync] 
Then you just have to place this shortcode at the page where the schedule from Add Classes plugin his.

Now when a use schedules a new class a button to add the event to Google Calendar will show.

 
== Installation ==
Simply drag and drop the plugin .zip via back-office or 
 
1. Upload plugins folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
